This site should be added to trusted zone, in order to work properly.
To add this site to trusted zone folowing steps should be performed:
Go to Tools-Internet Options-Security
Click the "Security" tab and choose the "Trusted Sites" icon
Then click on the button "Sites". Be sure to remove the check by the entry "Require server verification (https:)...."
Enter this site url in the line provided, then "Add" button.
To get more details, on IE 6 security settings see this page

One more thing you should to do, while activex aren't signed with valid sertificate, is to allow download unsigned ActiveX controls: Go to Tools-Internet Options-Security
Click the "Security" tab and press custom level button
Select "prompt" for "download unsigned ActiveX controls" item"
